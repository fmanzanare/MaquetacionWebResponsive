Generics en Java

Java nos ofrece una gran utilidad denominada Generics, que permite que las clases y sus objetos tenga mucha más útilidad.

La base de este concepto es que una clase admita cualquier tipo de dato para trabajar con él. Es decir, que el objeto que creemos de una clase que admita Generics, no permitirá crear un objeto de esa clase con el tipo de dato que deseemos.

Un claro ejemplo de esta propiedad, está implementada por los conjuntos (Set, Map, ArrayList, List, etc.) ya que la misma clase nos permite crearlos del tipo que deseemos:

		Set<Integer> set1 = new HashSet<>();
		Set<String> set2 = new HashSet<>();
		Set<"ObjectX"> set3 = new HashSet<>();
